# 古诗词赏析

一个基于 React + Node.js 的古诗词赏析 Web 应用，集诗词阅读、AI 创作、用户系统与收藏管理于一体。

## 功能特性

### 诗词赏析
- 收录 20 首经典古诗词与文言文（含高中必背 7 篇、初中必背 7 篇）
- 每首诗词配有完整译文、创作背景、诗人故事
- 支持按标题、作者、内容全文搜索
- 自动区分诗词与文言文，采用不同排版样式

### 诗词创作
- **智能生成**：输入灵感描述，AI 自动生成 3 首风格相似的诗词
- **手动创作**：选择诗词类型后自由创作，附带格律指南
- 支持 6 种体裁：五言绝句、七言绝句、五言律诗、七言律诗、词（长短句）、自由体诗
- 每种体裁提供格式说明、押韵规则、示例与创作提示

### 用户系统
- 本地注册 / 登录（基于 localStorage）
- 登录状态持久化，刷新不丢失
- 个人中心页面展示头像、收藏统计

### 收藏管理
- 在诗词详情页一键收藏 / 取消收藏
- 创作完成后可保存到"我的创作"
- 个人中心分标签查看：收藏诗词 / 我的创作
- Toast 通知反馈操作结果

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端 | React 18 + Vite 5 |
| 后端 | Node.js + Express 4 |
| 样式 | 纯 CSS（响应式设计） |
| 数据存储 | JSON 文件（后端）+ localStorage（前端） |

## 快速开始

### 环境要求

- Node.js >= 16
- npm >= 8

### 安装与运行

**1. 启动后端服务（端口 3001）**

```bash
cd backend
npm install
npm start
```

**2. 启动前端开发服务器（端口 5173）**

```bash
cd frontend
npm install
npm run dev
```

**3. 访问应用**

打开浏览器访问 http://localhost:5173/

## API 接口

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/poetry` | 获取全部诗词列表 |
| GET | `/api/poetry/:id` | 获取单首诗词详情 |
| GET | `/api/poetry/search/:term` | 搜索诗词 |
| POST | `/api/generate` | AI 生成诗词 |

### POST /api/generate 请求示例

```json
{
  "type": "七言绝句",
  "content": "秋天的夜晚，独自在异乡看着月亮，想念远方的家乡和亲人",
  "author": "笔名"
}
```

## 页面说明

| 页面 | 路由状态 | 说明 |
|------|----------|------|
| 首页 | `home` | 诗词卡片列表，支持搜索 |
| 诗词详情 | 选中诗词 | 展示全文、译文、创作背景、诗人故事，可收藏 |
| 创作页 | `create` | 选择体裁 → 智能生成 / 手动创作 |
| 登录 / 个人中心 | `profile` | 登录注册、查看收藏与创作 |
